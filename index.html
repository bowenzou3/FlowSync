🚦 How to Use

Getting Started

1. Select an algorithm from the control panel

2. Adjust traffic parameters as needed

3. Click "Start" to begin the simulation

Algorithms Explained

  • Fixed Time: Traditional signal timing (30s cycles)
  • Adaptive: Adjusts timing based on real-time traffic density
  • Q-Learning AI: Uses reinforcement learning to optimize signals
  • Max Pressure: Prioritizes directions with highest queue pressure

Scenarios

  • Normal: Balanced traffic flow
  • Rush Hour: High volume from main directions
  • Emergency: Emergency vehicle prioritization
  • Accident: Lane blockage simulation
